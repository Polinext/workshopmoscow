# VIBE/CODE — сайт интенсива

Готовый статический сайт на HTML, CSS и JavaScript. Его можно бесплатно
опубликовать через GitHub Pages или подключить к DigitalOcean App Platform.

## Файлы

- `index.html` — содержимое и структура сайта;
- `styles.css` — дизайн и мобильная адаптация;
- `script.js` — вкладки, прогресс чтения и кнопки копирования;
- `assets/` — изображения, используемые на сайте;
- `package.json` — минимальная настройка для хостинга, если платформа просит Node-проект;
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages.

Важно: загружайте папку `assets` целиком, не меняя названия файлов внутри.

## Вариант 1. Опубликовать через GitHub Pages

1. Создайте на GitHub новый репозиторий.
2. Загрузите **все файлы из этой папки в корень репозитория**.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите:
   - Source: `Deploy from a branch`;
   - Branch: `main`;
   - Folder: `/ (root)`.
5. Нажмите `Save` и подождите несколько минут.

Ссылка будет выглядеть примерно так:

`https://USERNAME.github.io/REPOSITORY/`

GitHub Pages для такого статического сайта можно использовать бесплатно.

## Вариант 2. Подключить хостинг

1. Сначала загрузите проект в GitHub.
2. В TimeWeb Cloud, DigitalOcean или другом хостинге создайте статический сайт.
3. Выберите GitHub и нужный репозиторий.
4. Укажите ветку `main`.
5. Проверьте, что ресурс определён как `Static Site`.
6. Если платформа просит настройки:
   - Build command: `npm run build` или пусто;
   - Publish / output directory: `.`;
   - Start command: `npm start`, если хостинг требует команду запуска.
7. Нажмите `Deploy`.

Хостинг будет автоматически публиковать новую версию после изменений
в ветке `main`, если включён автодеплой.

## Подключение своего домена

Домен не нужно добавлять в файлы заранее.

1. Купите домен.
2. Добавьте его в настройках проекта в GitHub Pages или DigitalOcean.
3. Скопируйте предложенные сервисом DNS-записи.
4. Добавьте эти записи в панели регистратора домена.

После выбора конкретного домена можно добавить файл `CNAME`.

## Локальный просмотр

Откройте `index.html` в браузере или запустите любой локальный статический
сервер из этой папки.
